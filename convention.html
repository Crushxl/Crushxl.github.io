Airspace空域 玩家公约 (2020 Ver.1)

公约声明

为保证提供更好的游戏环境、保护空域玩家的权益，您需要遵守并同意以下公约：

  1. 关于处罚：玩家违反空域公约中的任何条例时，管理组有权给予玩家不同的处罚。包括赔偿、禁言、惩罚标记、临时封禁、永久封禁、联合封禁等。当玩家被联合封禁时，与我们订阅/合作的所有服务器也会将玩家一并封禁。此举旨在增加处罚的威慑力。
  2. 关于修改： 本公约中的内容会依据实际情况不断调整，更改内容将会在空域交流群中以公告形式推送，恕不另行通知。
  3. 关于阅读：玩家决定使用空域Airspace服务器/交流群/动态时，便代表其已阅读、理解并愿意遵守公约中所有内容。如有疑问，可询问管理组成员，空域管理组保留最终的解释权利。

 

正文内容

注意：正文部分仅针对空域Airspace 纯净生存子服 有效，其他服务器请参照这里（未编辑）。


I.申请资格

  1. 玩家在申请加入服务器时应先在官方网站上提交申请表。
  2. 玩家应该确保申请表中的内容真实、有效。
  3. 单个玩家不应填在上一份申请的审核期内再次填写申请表。
  4. 自2020.07.01日起，空域将开启正版验证。届时已有离线模式账号无法登入服务器。若希望入正转移财产，请联系管理组。

II.服务设施

  1. 玩家不应破坏空域的任何服务器设施，例如攻击服务器、官网（包括但不限于DDOS攻击/ping攻击/压测）等行为。由此造成的经济损失均由攻击者承担，必要时依法追究其法律责任。

    2. 违反处罚：Uniban联合封禁。

  2. 空域在搭建时已经极力修复游戏平衡的漏洞、危害服务器安全的BUG，但我们仍无法避免第三方插件、更新带来的问题，发现此类问题，玩家应该通过官网的“反馈建议”进行提交，而不是恶意利用或牟取不正当利益。
    3. 违反处罚：封禁3-7天，情节严重者永久封禁。

III.建筑相关

  1. 禁止大规模破坏或者恶意破坏自然地形。
  2. 禁止建造、使用超频的红石机械。（1tick内激活大于1次，包括1次）
  3. 禁止制作对服务器内任何玩家等带有侮辱性的建筑。
  4. 建筑中不应带有多数人认为的色情、暴力、血腥的元素。
  5. 使用大规模破坏机械、大规模占地规划时，（轰炸机、大型建筑等）应事先在官网的“反馈建议”中提交申请，通过后允许实行。违反此项规定应该加倍处罚。（对于建筑，管理组无权拆除，但同样适用于VII.5中的条例）
    6. 违反处罚：禁言1-2天，情节严重者封禁2天。
  6. 避免搭建严重影响美观、不负责任的建筑（如高达256格的柱子），管理组有权要求玩家清除。

IV.补偿相关

  1. 在内测和公测下，玩家的所有赞助行为会被累计并保留，补全对应的金额享受不同权益。点击这里了解有关赞助的更多具体内容。
  2. 如需要永久退出服务器，可请联系管理组或通过官网提交建议。玩家的物品财产将会被立刻清除（不包括建筑等）。由管理组审核玩家的游戏行为（包括未完成的建筑/留下的安全隐患），完成检查后视作退出服务器，玩家不再受到本公约的约束，不享有之前存在的权利。
  3. 由于玩家自身失误（游戏内失误）、或服务器不可抗力的延迟、回档带来的损失，管理组不予受理。请留意公告/交流群中推送的维护公告。
  4. 由于玩家纠纷等引起的损失，理赔游戏内财产损失的，请携带证据（若有）通过官网的“举报玩家”提交申请。

V.财产相关

  1. 不同子服中的货币、物品互不相同，也无法回兑成现金。
  2. 玩家间进行的交易请在服务器内留下记录（私聊有效）。便于管理组在发生交易纠纷时快速判定责任。
  3. 不允许以桉树叶为媒介（如在讨论群内）进行任何形式使用现实中货币（包括但不限于使用网上交易媒介如支付宝、微信、QQ红包、PayPal等；包括但不限于使用 Bitcoin, Monero 等公认数字货币）的交易。
  4. 由于借用、盗取、转让、泄露密码导致正版账号发生的风险和财产问题，由玩家自行承担，空域不予受理。（内测期间可发起举报）
    5. 违反处罚：对于欺诈行为，永久封禁。

VI.言行举止

  1. 考虑他人感受，不给他人造成困扰。
  2. 不开受众面小的玩笑，以免他人误解。
  3. 避免辱骂、侮辱、凌辱，应做到言行文明。
  4. 禁止恶意在WEB卫星地图刷屏/服务器内私信刷屏/喊话刷屏（5连类似信息）。
  5. 禁止发表/讨论相关敏感话题。
  6. 禁止言语攻击他人的行为。
  7. 禁止鼓吹、煽动仇恨。
  8. 禁止发表色情、暴力、血腥的内容。
  9. 禁止人肉他人/侵犯他人隐私权利的行为。
  10. 对于他人的言论，应避免过度地钻牛角尖，刻意曲解含义。
    11. 违反处罚：禁言3小时-3天，情节严重者封禁1-5天，再次违反加倍处理。
  11. 禁止干扰空域管理组的正常管理行为。包括但不限于调查财产、调取私聊记录、ip追踪等，违反此条可酌情在原基础上追加处罚。
  12. 禁止在空域任何平台发表除空域相关内容以外的广告（包括但不限于其他服务器的游戏截图或服务器列表截图）、含误导的欺诈信息、违反国家法律的信息。
    13. 违反处罚：对于广告，封禁1-7天，再次违反永久封禁。

VII.游戏规定

  1. 空域服务器禁止玩家在主世界、下界和末地内跑图超过±100 000米或恶意跑图消耗服务器算力。
  2. 玩家不应当在±20 000米外建筑，如有发现，保留管理组无条件拆除的权利，不予补偿。
  3. 空域原则上尊重玩家行为，不对红石/高空流水/实体堆积做出过多限制，但玩家不应恶意堆叠消耗服务器算力。
    4. 违反处罚：禁言12小时，并要求整改。
  4. 玩家不应当在游戏内恶意破坏玩家建筑，例如放置拆除，破坏周边自然地形等擦边行为。
  5. 若希望保护规划自己的建筑，请在建筑周围设置至少3处醒目标示，提醒玩家不要占地。若玩家在上述情况下仍然占地的，视作违规。
  6. 玩家不应当在未经允许或明确告知私有的情况下取用任何玩家的资源。（空域主城内财产开放）
  7. 除处罚外，任何玩家、管理组不应当干涉、阻止玩家的正常游戏，包括但不限于移动（参观）、放置破坏（建筑，小镇规则不适用于此，参见具体条目）、聊天、交互生物（例如村民交易）、容器（例如查看箱子，若不想公开箱内物品，请说明或压箱）。
  8. 使用修改的客户端、资源模组、第三方修改器、计算器等任何破坏游戏平衡的软件。包括但不限于地图种子计算器、附魔种子计算器、自动钓鱼、挂机辅助、飞行、透视、影响pvp公平（漂浮、自瞄、无击退等）。
    9. 违反处罚：根据举报者的原谅态度（至少禁言1小时）和情节严重程度处以禁言1小时-7天/封禁1-28天/Uniban联合封禁。涉及财产的，应赔偿或者拆除建筑。

VIII.小镇规定

  1. 新玩家（自审核通过起）在24小时内无法加入任何小镇，请等待冷却期结束。
  2. 目前只支持赞助形式的新建小镇，赞助新建请访问赞助链接了解详情。
  3. 对于小镇如何加入，不同的小镇有不同的规则，具体请询问游戏内成员、镇长或私信管理组。
  4. 不同小镇间成员应当友好相处，玩家没有权利在未加入的小镇中破坏、放置、干扰玩家正常活动。
  5. 若要退出小镇，请通过官网的"反馈建议"进行，我们会通知相关镇长核实操作。
  6. 在不违反Airspace空域 玩家公约的前提下，小镇镇长和小镇管理员保留对其区划内的管理权利。
    7. 违反处罚：查核后由管理组和镇长讨论决定。

IX.称号说明

  1. 称号是标识玩家对空域的贡献/赞助和付出的肯定。任何人不能诋毁，或通过第三方手段添加称号。
  2. 称号分别有5种获取形式，最多选择叠加4个显示：
      • 2.1 申请加入小镇成为其成员的，联系管理组申请。样式："无名镇"。
      • 2.2 对空域或小镇做出巨大贡献的，本称号定期授予，无需申请。样式："建筑师"。
      • 2.3 完成Airspace空域内的挑战项目，联系管理组申请，样式："XXX"。
        • 2.3.1 全成就：完成原版的所有成就。（不包括隐藏）
        • 2.3.1 活跃成员：累计在线时长达到720小时（即将开放）
      • 2.4 完成Airspace空域不定期举办的活动，详见服内公告通知。样式："XXX"。
      • 2.5 对空域赞助达到相应金额，提供资金支持的（详见赞助链接）。样式："自定义" 。
  3. 每个称号长度不能大于8个字符，总显示称号长度不大于32字符。显示优先级为小镇|荣誉|挑战|活动|赞助。
  4. 以上称号，除活动外长期有效。长期不活跃玩家（超1月未上线）管理组有权利收回其荣誉称号。

X.事件处理

  1. 处罚除非有充足证据证明判断有误，否则不予撤销。点击这里提交申诉。
  2. 若被举报者拒不配合调查或拒不接受、逃避处罚，可酌情加大处罚力度。
  3. 对于被实施永久封禁的玩家，空域管理组可视情况决定将之移出群聊或移除群聊的同时加入黑名单。
  4. 若处理办法中出现歧义，按照对权益被损害方有利的解释执行。
